# Donjons_et_Dragons_Java
Le Projet Donjons et Dragons vise à l'apprentissage de Java.
Le but est de créer un jeu avec des règles prédéfinis en codant le jeu en programmation orienté object.

## Les compétences à développer:
- Déduction des classes à créer en suivant les règles du jeu
- Création d'un diagramme de classe UML avec les propriétés, méthodes etc...
